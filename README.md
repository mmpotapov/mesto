# Проектная работа «Mesto: валидация форм и попапы»
## Яндекс.Практикум — Спринт 6
------
Адаптивный одностраничный сайт с постепенным введением дополнительного функционала.

На данном этапе **используется**:\
• Вёрстка с использованием Flexbox и Grid.\
• Адаптивность под экраны мобильных устройств.\
• Оверлеи и формы.\
• Добавление DOM-элементов с помощью JavaScript.\
• Реакция на различные действия пользователя.\
• Редактирование данных на странице.\
• Валидация всех форм на сайте.
